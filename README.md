## Instalacion

1. Clona el repositorio
   ```sh
   git clone https://github.com/NachoCencosud/tweets.git
   ```
2. Instala las dependencias con npm
   ```sh
   npm install
   ```
3. Corre la aplicacion en modo dev
   ```js
   npm start
   ```
